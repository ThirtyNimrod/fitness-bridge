# src/

This directory contains the main source code for the Fitness Bridge AI application.

## Subdirectories

- **clients/**: API client classes for integrating with external services (Hevy, Fitbit).
- **graph/**: LangGraph-based AI agent implementation for conversational coaching.
- **tools/**: LangChain tools for fetching and processing fitness data.
- **utils/**: Utility modules, including database management for chat history.

## Files

- **__init__.py**: Package initialization file.
