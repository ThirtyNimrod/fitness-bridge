# clients/

This directory contains API client classes for integrating with external fitness services.

## Files

- **hevy_client.py**: HevyClient class for interacting with the Hevy API. Handles authentication, fetching workout routines, retrieving recent workouts, and updating routines.
- **fitbit_client.py**: FitbitClient class for interacting with the Fitbit API. Manages OAuth2 tokens, fetches user profile data, sleep summaries, and heart rate information for readiness assessment.
- **ReadME.md**: This documentation file.
